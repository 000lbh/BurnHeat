1. 卡死了怎么办？
    把线拔了，程序恢复，你可以保存数据。线一定要插牢！
2. 读到损坏的数据怎么办？
    再读一遍
3. 无法连接怎么办？
    打开设备管理器，找到串口设备，将串口名称（COM#或者其他）粘贴过来。另外记得装驱动。
4. Mac版本？
    老师也没有给驱动，做出来也没啥用（不过类Unix系统可以读/dev下的文件）
5. 数据丢失？
    抱歉。
6. 编写语言？
    Object Pascal
7. 其他问题？
    可以提，但是时间原因，未必能一一解答。
